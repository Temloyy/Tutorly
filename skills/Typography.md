Typography Rules

This document defines the typographic system used throughout this application

The system follows the principles influenced by the Material Design typography scale

Objectives

Typography provides:

Clear hierarchy
Consistent reading rhythm
Comfortable scanning
Font family

Primary UI Font: Google Sans

Fallback fonts:

-apple-system
system-ui
sans-serif
Arial
Roboto
Type scale

Use the type scale in the design tokens

Usage guidelines
Page display text should use Display Large
Section titles should use Headline Large
Button labels should use Label Large
Form labels should use Label Medium or Label Large
Standard paragraphs and body should use Body Large
Card titles should use Title Medium
